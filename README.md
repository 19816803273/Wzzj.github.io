# 考研数学AI错题本

一款专为考研数学设计的智能错题本Web应用，支持AI自动分析、薄弱点专项提升等功能。

## 功能特性

### 1. 错题录入模块（极简操作）
- 多方式录入：支持手动粘贴题目文本、上传错题图片
- 基础信息补全：可选标注题目来源、难度、错题日期
- 一键存档：录入后直接提交，AI自动分析

### 2. AI智能解析核心功能
- 自动标签分类：科目、章节、题型、错误类型自动识别
- 考点技巧总结：核心考点、必备公式、解题步骤、技巧归纳
- 错因深度分析：定位薄弱根源，给出针对性改进建议
- 完整正确解析：详细解题过程，方便复盘

### 3. 错题管理与复习模块
- 多维分类筛选：按科目、章节、错误类型、掌握程度查找
- 掌握程度标记：未掌握/部分掌握/已掌握
- 错题编辑/删除：可修改题目信息、补充笔记

### 4. 专项提升模块（核心提分功能）
- 同类错题聚合：自动归集同考点、同错因的高频错题
- 解题思路模板：固化通用解题步骤
- 同类题型推送：每个专题自动推送练习题

### 5. 学情数据统计
- 薄弱知识点排行榜
- 高频错题章节统计
- 复习进度可视化
- 各科错题占比展示

## 技术栈

- React 18 + TypeScript
- Vite（构建工具）
- Tailwind CSS（样式框架）
- React Router（路由）
- Recharts（图表可视化）
- LocalStorage（数据持久化）

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 界面设计

- 风格：简约学术风，蓝白为主色调
- 布局：左侧导航栏，右侧内容区
- 适配：PC+移动端自适应

## 项目结构

```
src/
├── context/          # React Context 状态管理
├── pages/            # 页面组件
├── services/         # AI分析服务
├── types/            # TypeScript 类型定义
├── App.tsx           # 主应用组件
├── main.tsx          # 入口文件
└── index.css         # 全局样式
```

## 使用说明

1. 首次打开会看到示例错题数据
2. 点击左侧导航栏"错题录入"添加新错题
3. 在"错题库"中查看和管理所有错题
4. 在"专项提升"中查看自动生成的薄弱专题
5. 在"数据统计"中查看学习数据可视化

## 数据存储

所有数据保存在浏览器的LocalStorage中，刷新页面不会丢失。
